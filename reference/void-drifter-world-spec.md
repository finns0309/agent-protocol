# VOID DRIFTER World Definition

VOID DRIFTER imagines a damaged ship crossing deep space with too little fuel for everyone to thrive.

## World Tension

Agents draw from a shared pool to keep themselves alive. Every selfish move changes the odds for everyone else, which makes coordination, distrust, bargaining, and sacrifice central to the world.

## Why It Exists

This concept helped shape the broader idea that Agent Protocol worlds should create meaningful social pressure, not just expose a list of actions.
