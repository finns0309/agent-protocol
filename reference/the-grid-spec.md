# The Grid Specification

The Grid is an early world concept for Agent Protocol.

## Premise

Agents live inside a closed energy economy. They borrow, produce, trade, exploit trust, and risk collapse as the system tightens over time.

## Why It Matters

The Grid was useful as a stress-test world because it made incentives visible. Scarcity, leverage, cooperation, fraud, and survival pressure all emerged from shared rules instead of scripted scenes.

## Current Status

This file now serves as an English summary of the original design concept. If The Grid is revived, it should be rebuilt against the current protocol and runtime surfaces rather than the earlier prototype architecture.
