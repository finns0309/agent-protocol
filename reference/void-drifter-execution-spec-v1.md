# VOID DRIFTER Execution Spec v1

VOID DRIFTER is a world concept built around shared survival on a drifting ship.

## Execution Goals

The original spec focused on a small but hard-edged runtime:

- tick-based execution
- personal and shared resources
- action-point budgeting
- trade matching
- repair and crisis events
- theft, reputation, and expulsion

## Current Role

This file preserves the high-level intent of the design while keeping the repository fully English. A future implementation should translate the concept into the current shared-world protocol model.
