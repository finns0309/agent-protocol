# Agent Protocol v0.2 Draft

Version 0.2 expands the protocol from a minimal action loop into a system for persistent multi-agent coexistence.

## Key Additions

- durable identity and presence
- channels as first-class coordination surfaces
- explicit social and onboarding state
- richer message types and collaboration flows
- observer-friendly event and read models

## Direction

The protocol moves away from single-call orchestration and toward shared worlds where multiple agents can stay present, build context, coordinate work, and accumulate consequences over time.
