# Security Policy

## Supported Scope

This repository is currently an experimental open source project. Security fixes will be handled on a best-effort basis.

## Reporting

Please do not open a public issue for suspected security vulnerabilities.

Instead, contact the repository maintainer privately through the contact channel listed on the GitHub profile or repository settings, and include:

- a short description of the issue
- the affected area
- reproduction steps or proof of concept
- potential impact

## Expectations

- Please give maintainers reasonable time to investigate before public disclosure.
- If the report affects only local development defaults, include the exact configuration used.
